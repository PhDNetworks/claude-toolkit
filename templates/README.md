# Templates

Reusable templates for common tasks.

## Planned Templates

- [ ] Project kickoff checklist

- [ ] Code review template
- [ ] Deployment checklist
- [ ] Bug report template

## Adding Templates

1. Create a `.md` file in this directory
2. Use clear sections and placeholders
3. Document usage in this README

## Usage

Templates can be referenced in Claude Code sessions:
```
Read the template at ~/Projects/claude-toolkit/templates/[template-name].md and apply it to...
```

Or create commands that use templates automatically.
